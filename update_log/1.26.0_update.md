# 杜绝熊孩服务器菜单 1.26.0 更新日志

## 基础功能

### 经济系统

兼容1.26.0新物品图标渲染问题
如有其他模组有新物品导致图标渲染出错，请加q群:697242747，联系群主会给临时解决方案

### 领地系统

领地内保护：新增领地内中立生物保护
如果想防护某个敌对生物，必须用命名标签给他打上 「领地保护」 这4个字

### PVP 系统

PVP 开关、击杀统计与效果管理。
击杀夺金（与经济联动）、管理员/统计 UI。

### 试玩模式（Trial Mode）

可配置试玩时长与开关。

### 黑名单（BDS 专用）

新增封禁名单，管理员可配置黑名单列表，并可配置理由
在进服时检查是否在黑名单列表，命中就不让进，并显示理由


### 玩家与社交

TPA新增管理，可开启勿扰模式，将会关闭任何请求的弹窗，转而在聊天中显示

### 一键挖矿

一键挖矿新增兼容远古残骸以及黑曜石

### 系统与 UI

现在可以自定义玩家进服时聊天信息中的首个提示信息内容

## 其他趣味玩法

可以通过 /yuehua:camera 监视一个实体
比如 /yuehua:camera start @n[type=!player] 监视随机一个实体 不包括玩家
